# 新东方云教室视频课程下载器（Win电脑版）

新东方云教室视频课程下载工具，一键下载新东方云教室上面已购买的付费课程

【主要解决课程过期后无法观看的问题，下载到本地可永久观看】

工具获取地址：https://pyjun.lanzoup.com/s/course-downloader

## 安装教程
从蓝奏云网盘下载最新版学无止下载器后，点击确认一路安装即可

【如果电脑安装杀毒软件，建议关闭杀毒软件后再下载安装，以免误报拦截】
